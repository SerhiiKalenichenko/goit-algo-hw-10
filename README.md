# Завдання 2 — Метод Монте-Карло

Обчислено визначений інтеграл функції f(x) = x² на відрізку [0; 2] методом Монте-Карло (N = 1 000 000 випадкових значень).

Порівняно з аналітичним значенням інтеграла:
Монте-Карло дає результат, близький до 2.666…
Аналітичне значення: 2.666666666666667
Похибка: мала, в межах статистичної дисперсії

## Висновок

Результат методу Монте-Карло практично збігається з аналітичним, що підтверджує правильність реалізації алгоритму та коректність обчислень без використання SciPy.
